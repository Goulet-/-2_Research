Research:

Color Managment:
The primary goal of color management is to obtain a good match across color devices; for example, the colors of one frame of a video should appear the same on a computer LCD monitor, on a plasma TV screen, and as a printed poster.

Dot Gain:
Is a phenomenon in offset lithography and some other forms of printing which causes printed material to look darker than intended. It is caused by halftone dots growing in area between the original printing film and the final printed result.

Preflight:
This is the process of gathering all the information you will need for the current project you are working on. Review your files, such as the fonts, colors, layout, links/ graphics, and clean files.

Halftone Placement:
Where continous tone imagery contains an infinite range of colors or greys, the halftone process reduces visual reproductions to an image that is printed with only one color of ink, in dots of differing size   by amplitude modulation or spacing frequency modulation.

Gray Balance:
When you are correcting an image you have to consider the gray balance because if you do edit your image, your colors will look more natural. 

Classic Curve:
The classic curve or the S curve will help you improve the contrast of your photos. This is an image correcting tool as well as using the levels tool.

Gand Run Printing:
This is a process in printing when you want to print multiple projects onto on sheet of paper. This will help you save money in the long run. 

Prepress:
The term used in printing and publishing industries for the processes and procedures that occur between the creation of a print layout and the final printing.

InDesign:

Color Mode:
Use the CMYK mode when preparing an image to be printed using process colors. Converting an RGB image into CMYK creates a color separation. If you start with an RGB image, it's best to edit first in RGB and then convert to CMYK at the end of your editing process.

Curves:
The curves tool can take input tones and selectively stretch or compress them. A tonal curve is controlled using any number of anchor points. The result of a given curve can be visualized by following a test input tone up to the curve, then over to its resulting output tone.

grayscale image color:
If you swap an image to gray mode you can add tints to it dirictly in InDesign. A duotone is an image with two colors or tones, effectively a grayscale image, but more accurately one that substitutes a color for black and another color for white or negative space.

Crop vs. Reframing:
Cropping your image will help you cut out unwanted objects in your image. If you want to reframe your image you can modify it by changing its relationship to its frame.
